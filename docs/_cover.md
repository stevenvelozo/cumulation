# Cumulation

> A browser and Node.js client for a Retold-style RESTful data endpoint

- Read, create, update, and delete records for a configured entity
- Single-record and filtered plural reads with paging
- Automatic cookie and header management per request
- Graph query layer that resolves joins and builds FoxHound filter URLs
